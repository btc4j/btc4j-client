Bitcoin Java Client (btc4j-client)

Open source Java implementation of the Bitcoin client.

Using btc4j-client

btc4j-client is free software under The MIT License (MIT). It is maintained by Guillermo Gonzalez (email@ggonzalez.info).

Maven pom.xml dependency:

    <dependency>
        <groupId>org.btc4j</groupId>
        <artifactId>btc4j-client</artifactId>
        <version>0.0.3-SNAPSHOT</version>
    </dependency>
